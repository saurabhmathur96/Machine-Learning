# Linear-Regression
An implementation of [Linear regression](https://en.wikipedia.org/wiki/Linear_regression) with gradient descent.

## linregr.py
Performs linear regression on data in data.csv and plots the resulting linear model.

## Requirements
- NumPy
- Matplotlib
